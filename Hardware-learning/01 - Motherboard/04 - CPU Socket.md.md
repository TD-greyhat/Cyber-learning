The CPU socket is on the motherboard, this is where we put the CPU.

	- Physical fastening
	- power connection
	- data connection between CPU and the motherboard

- There is pins on the CPU or on the Socket.
	around 755-1718.
	- every pins has their own task
		- power supply
		- grounding
		- memory communication
		- PCIe communication
		- control signals

- Diversity of CPU compatibility
	- different psihyc
	- different amount of pins
	- different pin allocation
	- different electrical requeriments

- Socket types
	- LGA(Land Grid Array)
	 The pins in the sockets.
		adv: Safer for the Processors, no pin bending risk.
		disadv: Less safer for the socket.
	- PGA(Pin Grid Array)
	 Pins on the CPU.
	- BGA(Ball Grid Array)
	 The CPU is soldered onto the motherboard.
	    - can not simply change the CPU
	    - its in laptops, miniPCs, phones

- BIOS updates 
  example:
		There is new CPU ->
		The old motherboard socket is good for it ->
		But the BIOS can not recognize the new CPU ->
		BIOS updates necessary
